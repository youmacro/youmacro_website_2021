Download Any Video

YouMacro can download any video from your browser.

Simply drag links from your browser onto this app.
Links can appear as text, images or favicons at the left side of the address bar.

YouMacro is open source.
Source and binaries are provided for educational purposes only.

Source code is available at: GitHub
More node graph related code is available at: www.nodegraph.com

To install YouMacro, simply download and decompress the zip file.
